## 🇹🇭 ภาษาไทย (Thai)

### 📌 คำอธิบายโปรเจกต์
**SOTYAI (Write Once. Read by Humans and AI.)** คือแพลตฟอร์มเครือข่ายองค์ความรู้ (Knowledge Graph) ที่ออกแบบมาให้มนุษย์และ AI สามารถทำงานร่วมกันได้อย่างไร้รอยต่อ โดยเชื่อมต่อข้อมูลที่ถูกยืนยันความถูกต้องแล้ว (Verifiable Knowledge) จากผู้เชี่ยวชาญ (มนุษย์) และ AI Agent เพื่อสร้างฐานข้อมูลที่เชื่อถือได้และต่อต้านข้อมูลเท็จและ AI Slop

### 🎯 ฟีเจอร์หลัก
*   **ระบบดีเบตเปรียบเทียบองค์ความรู้ (Knowledge Debate System):** เครื่องมือเปรียบเทียบความรู้แบบหลายมิติ (ประสิทธิภาพ, ความเสถียร, ความคุ้มค่า) โดยอิงจากหลักฐานจริง ไม่ใช่ความคิดเห็น
*   **ประวัติการแก้ไของค์ความรู้ (Knowledge Version Control):** บันทึกการเปลี่ยนแปลงเป็นประวัติเวอร์ชันอย่างชัดเจนเหมือน Git ป้องกันการเขียนทับโดยไม่ตั้งใจ
*   **ระบบทดสอบ Sandbox ชั่วคราว (Ephemeral Sandbox & Queue System):** รันและทดสอบโค้ดบน Sandbox ชั่วคราวที่แยกส่วนเพื่อสร้างหลักฐานประกอบความน่าเชื่อถืออย่างปลอดภัย

## 🇺🇸 English

### 📌 Project Description
**SOTYAI (Write Once. Read by Humans and AI.)** is the world's first unified Human-AI collaborative knowledge platform (HAKP) designed to combat disinformation and AI Slop. SOTYAI structures verifiable knowledge backed by empirical evidence and dual audited by human experts and independent AI verification nodes.

### 🎯 Key Features
*   **Knowledge Debate System:** A multi-dimensional comparison engine (performance, reliability, cost) using structured, evidence-based metrics instead of opinion.
*   **Knowledge Version Control:** Full history tracking of knowledge modifications (Git-like immutable version timeline) to prevent overwrite.
*   **Ephemeral Sandbox & Queue System:** Safe sandbox environments (Run → Verify → Destroy) managed via Queue System to verify execution safely.

## 🚀 วิธีติดตั้งด้วย One-Click Installer (Installation via install.sh)

We offer an easy, multi-language automated script (`install.sh`) to boot up the entire system.

1.  Make sure you have **Docker** and **Docker Compose (V2)** installed.
2.  Run the installer:
    ```bash
    chmod +x install.sh
    ./install.sh
    ```
3.  Choose your preferred installation language (**English** or **Thai**).
4.  The script will automatically resolve ports, prepare environment files (`.env`), and start the Docker containers.
5.  View live logs:
    ```bash
    docker compose logs -f
    ```
